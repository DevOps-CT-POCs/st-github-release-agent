## Template Structure

When generated, this file contains:

- **Repository**: Your GitHub repo (`owner/repo`)
- **Workflow**: The deployment workflow name
- **Latest successful runs**: Run IDs, timestamps, and deployed commit SHAs
- **Changes**: List of commits/PRs included since the last distinct deployment
- **Compare link**: GitHub compare URL between deployments
